The pages here described an earlier version of the FoodTruckNerdz web experience. Do **not** use them as a setup guide for new work; expect outdated tooling, URL patterns, and feature descriptions.

== Historical summary

FoodTruckNerdz is a service to help people find food trucks and save favorites. A companion app for Android and iOS let operators report location with a check-in and background monitoring during service hours. The *ftn-site-old* material reflected that era of the public website, before the product moved to the current *ftn-site* implementation.
